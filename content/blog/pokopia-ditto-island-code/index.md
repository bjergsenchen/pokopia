各位開拓中的百變怪可以在 Pokémon Pokopia 遊戲內輸入代碼「**PXQC G03S**」去參觀官方打造的島嶼喔🏝️

## 使用方式

1. 進入《Pokémon Pokopia》遊戲。
2. 開啟代碼輸入介面。
3. 輸入代碼 `PXQC G03S`。
4. 確認後即可前往官方島嶼參觀。
